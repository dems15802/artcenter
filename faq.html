자주하는 질문

고객님이 자주 하시는 질문을 모아 놓은 공간입니다.
보다 상세하고 신속한 답변이 필요하신 경우 1:1문의를 이용하시거나 전화문의 바랍니다.
충무아트센터 ARS : 02-2230-6600 (월~금 09:30~18:30 / 점심시간 12:00~13:00)
* 주말 및 공휴일 휴무

  • 선예매란?

    선예매는 충무아트센터 홈페이지에서, 홈페이지 회원만 가능한 예매입니다. 공연별로 시기의 차이는 있지만 일반예매보다 우선해서 예매가 가능합니다. 선예매 안내는 홈페이지 팝업을 통해 공지 됩니다.

  • 선예매 방법은?

    충무아트센터 홈페이지 회원이면 누구든 선예매가 가능합니다. 유료멤버십이 따로 있지 않습니다. 안내된 선예매 시간에 홈페이지에 접속하셔서 예매하시면 됩니다. (예매는 모바일 환경에서 불가능합니다)

  • 예매방법

    충무아트센터 홈페이지는 모바일 환경에서의 예매 시스템이 적용되어있지 않습니다. 인터파크 어플 혹은 컴퓨터를 통한 예매 부탁드립니다.

  • 예매확인

    충무아트센터 홈페이지에서 예매했을 경우 충무아트센터 홈페이지 내에서만 예매내역 확인 가능합니다. 인터파크에서 보이지 않더라도 이는 오류가 아니며, 충무아트센터 홈페이지 > 마이페이지를 확인 해주시기 바랍니다.

  • 결제수단 변경

    예매하신 공연에서 결제수단을 변경하고자 하시면 기존 예매건을 취소하고 원하시는 결제수단으로 다시 예매를 해 주셔야 합니다.
    ※ 다만 기존 예매건 취소 시 예매대기 서비스 등으로 인하여 동일좌석 재예매가 어려울 수 있음을 양지하여주시기 바랍니다.

  • 할인율 변경

    예매하신 공연에서 할인율을 변경하고자 하시면 기존 예매건을 취소하고 원하시는 할인율 선택 후 다시 예매를 해 주셔야 합니다.
    ※ 다만 기존 예매건 취소 시 예매대기 서비스 등으로 인하여 동일좌석 재예매가 어려울 수 있음을 양지하여주시기 바랍니다.

  • 환불계좌 불일치

    환불계좌 불일치의 경우는 아래 두 가지 경우입니다.
    1. 환불계좌를 잘못 입력하는 경우
    2. 본인명의의 계좌를 입력하지 않는 경우 고객님께서 직접 고객센터에 환불계좌 재접수 해주셔서 환불 정보를 수정해 주셔야합니다. 인터파크 고객센터 1544-1555 입니다.

  • 티켓 수령방법 변경

    인터파크 고객센터 1544-1555에서 가능합니다. 배송은 14일전까지만 배송 신청 가능하며, 할인 받은 예매건 또는 공연특성에 따라 배송 접수가 불가할 수 있습니다. 티켓 배송요금이 있으니 참고 부탁드립니다.

  • 개명으로 인한 고객정보 변경

    개명으로 인한 고객님 개인정보 변경은 QNA 페이지에 변경하시고자 하는 성함 알려주시면 변경 도와드리겠습니다.

  • 신뢰할 수 있는 사이트 설정

    충무아트센터 홈페이지 내 마이페이지 접속 시 ‘신뢰할 수 있는 사이트’ 관련 문구가 나오게 되면
    Internet Explorer 실행 → 도구 → 인터넷옵션 → 보안
    신뢰할 수 있는 사이트를 선택 → 보안수준 최소 → 사이트를 선택 →
    *.interpark.com을 입력 한 뒤, 하단 체크 박스를 끄고 추가 해줍니다.
    다음 인터넷옵션 → 도구 → 호환성 보기 설정에 들어가 interpark.com 추가 부탁드립니다.
